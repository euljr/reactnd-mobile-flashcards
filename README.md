# Mobile Flashcards

Este é o último projeto do Nanodegree Desenvolvedor React da Udacity, nele foi desenvolvido uma aplicação simples de flashcard.

## Rodando a aplicação

Neste projeto, utilizei o `yarn` em vez do `npm`, então para dar início ao desenvolvimento é só executar: 

```bash
 $ yarn install
 $ yarn start
```

Será iniciado ambiente do `Expo`, a partir de lá é possível iniciar a aplicação no simulador ou no aplicativo do Expo no Android ou iOS.

## Storage

Nesse projeto, optei por utilizar o `redux-persist`, no react-native ele utiliza o `AsyncStorage` por padrão e salva/recupera o state automaticamente.
